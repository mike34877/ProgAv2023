Necesitamos los plugins de javascript y typescript
npm install typescript

para compilar podemos usar
./node_modules/type_script/bin/tsc

Si queremos ir codeando y que se compile solo
./node_modules/type_script/bin/tsc -w

Estos dos ultimos comandos los podemos ejecutar desde el menu Execute Run Build Task... from the global Tasks menu.

para ejecutar
node Ctrl+F5

para debaguear
Presiona F5
